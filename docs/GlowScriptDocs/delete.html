Deleting an Object

To hide a GlowScript object just make it invisible: ball.visible = False. This does not delete the object from computer memory, and you can make it visible again later. (In JavaScript say ball.visible = false.)

If however you later re-use the name ball, for example by creating a new object and naming it ball, the computer will be free to release the memory used by the object formerly named ball (assuming no other names currently refer to that object). If the object is visible when you re-use the name ball, the original object will not be deleted from computer memory, and it will remain visible in the window.
